# Genesis 5 Website

A single-page React application for the Genesis 5 gaming event by GDXR Club, built with React and Tailwind CSS.

## Features

- **Hero Section**: Eye-catching landing with Genesis 5 branding
- **About Section**: Event details and competition stages
- **Prize Pool**: Visual display of prizes for top 3 winners
- **Memories**: Gallery section for past events
- **Smooth Navigation**: Fixed navbar with smooth scrolling

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The application will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Technologies Used

- React 18
- Tailwind CSS
- Vite
- Google Fonts (Londrina Solid, Livvic, Cairo, Mea Culpa, Inter)
